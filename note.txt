         create a tsconfig.json file
{
"compilerOptions": {
"target": "ES2020",
"module": "commonjs",
"rootDir": "src",
"outDir": "dist",
"strict": true,
"esModuleInterop": true,
"forceConsistentCasingInFileNames": true,
"skipLibCheck": true
},
"include": ["src"]
}

add this to the package.json
"scripts": {
"dev": "nodemon --watch 'src/**/*.ts' --exec ts-node src/server.ts",
"build": "tsc",
"start": "node dist/server.js"
}

create src folder

- create server.ts file to start the server 

- create app.ts file for the express app, middleware and routes

- create db.ts file for for DB access

- create routes folder then product.routes.ts to handle specific routes

- create controllers folder then product.controller.ts file to handle CRUD operations

- create validation folder then product.schema.ts file for Joi schemas create & update

- create middleware folder then errorHandler.ts to handle errors
